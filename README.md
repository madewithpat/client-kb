# Knowledge Base

A personal knowledge base powered by [Obsidian](https://obsidian.md) and [Claude Code](https://claude.ai/download).

## What it does

- **Auto-context:** Mention a client or project and Claude loads their full history
- **Auto-logging:** After meaningful work, Claude logs what was done and why
- **Meeting notes:** Share a transcript and get structured notes, action items, and follow-up emails
- **Weekly reviews:** Friday summaries of work across all your clients/projects
- **Daily journal:** Cross-project continuity between sessions

Works for anyone — freelancers, consultants, developers, designers, project managers, lawyers, anyone managing ongoing work.

## Getting started

1. Clone this repo (or use it as a template)
2. Install [Obsidian](https://obsidian.md) and open this folder as a vault
3. Open this folder in [Claude Code](https://claude.ai/download)
4. Say anything — the setup wizard starts automatically

The wizard asks your name, what you do, and what tools you use. Quick setup takes 2 minutes. Custom setup lets you connect Google Calendar, meeting recorders, and task managers.

## What's inside

```
CLAUDE.md               — Automation rules Claude follows
_templates/             — File templates (profiles, strategies, meeting notes)
SOPs/                   — Workflows (meeting processing, weekly review)
_setup/                 — Setup wizard and permissions
_daily/                 — Daily session journals
```

## Requirements

- [Claude Code](https://claude.ai/download) (free or paid)
- [Obsidian](https://obsidian.md) (free)

## Optional integrations

Connected during setup if you have them:

- **Google Calendar** — meeting detection and prep briefs
- **Meeting recorder** (Fireflies, Otter, etc.) — auto-process recordings
- **Task manager** (Todoist, etc.) — create tasks from action items
- **Claude in Chrome** — browser automation for fetching transcripts
